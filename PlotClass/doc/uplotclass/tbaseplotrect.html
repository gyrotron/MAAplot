Description

A Plotrect is the container for axes and series belonging together. Usually a plotrect will hold 2 axes and a 2D series or 3 axes and a 3D series. One or more Plotrect are placed into a TPlot. The Plotrect does some size calculations to fit the dataplot into the desired are. Therefore the Plotrect can be reagrded as the container class for the "plot" you get at the end with axes, captions and data plotted into. Note: More than one plotrect can be placed into a TPlot container class. The most common situation however would be to have one plotrect in one TPlot container. Do not use the base class - please use the TPlotRect class. Check TPlotRect for further functions and properties
