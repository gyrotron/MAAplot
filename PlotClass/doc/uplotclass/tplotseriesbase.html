Description

A Series (TPlotSeries) gets the data to be plottet (i.e. from the main application). Series (as implemented so far) have X and Y axis or X,Y and Z axis. The data is mapped to the screen for display by use of the axes defined. Do not use the base class - please use the TPlotSeries class. Check different series in uPlotSeries for further functions and properties.
