TPlotSeriesBase.FIsFastSeries

standard or fast series

Declaration

Source position: uplotclass.pas line 303

protected TPlotSeriesBase.FIsFastSeries : Boolean;

Description

The historic mode of operaqtion is draweing points directly onto the canvas of the TImage (PlotImage). Series working that way are TPlotSeries, TXYPlotSeries and TXYZPlotSeries. For these, IsFastSeries is false. The newer concept is drawing on a memory bitmap (TLazIntfImage) and overlay to the screen in TPLotRect.Redraw. This concept is significantly faster, therefore IsFastSeries = true for these series. Note that Redrawing significantly differs between fast and standard series. Fast and standard series cannot be mixed in ode plotrect.
