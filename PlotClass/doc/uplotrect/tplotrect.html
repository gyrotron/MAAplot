Description

The Plotrect is the main element containing exactly one dataplot. It determines the remaining size for axes after drawing a logend and a coloscale. The Plotrect contains also a TLazIntfImage (DataImage) where the fast series plot onto and overlays the dataimage onto the screen duriong redraw.
