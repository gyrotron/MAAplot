Description

When Data has changed (i.e. when CheckRefresh = true) also the markers need to be updated (PeakLists etc.). Markerupdate is also neccessary when markers are added, removed or their properties are changed. Failure to call UpdateMarkers will result in erroneous marker drawing as these are based on older data.
